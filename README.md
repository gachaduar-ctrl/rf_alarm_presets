## 🎉 RF Online Tools - Pitboss & War Scheduler v1.0.0

Aplikasi alarm untuk RF Online — schedule pitboss spawn & war events.

### Downloads
- **Windows **:  Check Release (Recommended)

### Features
- ⏰ Multi alarm dengan waktu manual
- 🔔 4 pilihan sound (Bell, War Horn, Siren, Beep)
- 🌐 Custom URL audio support
- 🐉 Auto-load preset dari GitHub
- 🎨 Dark theme
- 🔊 Alarm bunyi sampai di-dismiss

### Requirements
- Windows: WebView2 (biasanya sudah pre-installed di Win 10/11)
